# 🎓 EduNotes AI

> **Transform YouTube Lectures into Smart Study Material**

EduNotes AI is an AI-powered web application that transforms YouTube lecture videos into structured learning resources such as summaries, notes, transcripts, quizzes, flashcards, and downloadable PDF notes. It helps students learn more effectively by reducing the time spent on manual note-taking and making revision faster and more interactive.

## 📌 Problem Statement

Students spend a significant amount of time watching lengthy educational videos and manually preparing notes. This process is time-consuming, makes revision difficult, and provides no quick way to evaluate understanding after completing a lecture.

EduNotes AI addresses these challenges by automatically converting YouTube lecture videos into organized study materials using Artificial Intelligence.



# 💡 Our Solution

EduNotes AI enables users to paste a YouTube lecture URL and instantly generate:

* 📝 Structured Notes
* 📄 Lecture Transcript
* 🤖 AI Summary
* ❓ AI-Generated Quiz
* 🧠 Interactive Flashcards
* 📥 Downloadable PDF Notes

This allows students to focus on learning rather than spending hours creating notes manually.

# 🎯 Core Features

### 📝 Structured Notes

Automatically converts YouTube lecture transcripts into well-organized study notes for easy understanding and quick revision.

### ❓ AI Quiz Generator

Creates AI-generated multiple-choice questions that help users test their knowledge after watching a lecture.

### 🧠 Interactive Flashcards

Generates flashcards from important concepts to improve memory retention and make revision more effective.

### 📄 Transcript Extraction

Automatically extracts transcripts from YouTube lecture videos for AI analysis and content generation.

### 📥 PDF Export

Allows users to download generated notes as PDF files for offline learning and future reference.

### 📚 Dashboard & History

Displays generated study materials in an organized dashboard and stores previous sessions for easy access.
